# Reading Model Research Outline

These YAML files preserve the investigation structure used before the reading-model persistence
closure was written:

- `outline.yaml` lists the research objects and questions.
- `fields.yaml` defines the evidence fields collected for each object.

They are supporting evolution artifacts, not runtime configuration and not current implementation
requirements. Start with [`../persistence-closure.md`](../persistence-closure.md) for the resulting
engineering conclusions.
