# PaperLoom Interview Resources

## Knowledge

- [PaperLoom architecture overview](docs/architecture/overview.md)
  Repository source of truth for the live request path, responsibility boundaries, storage roles, and authorization ladder.
- [Reading Model and Agent tools](docs/architecture/reading-model-and-agent-tools.md)
  Use for the canonical paper model, request-local tool state, retrieval behavior, and final-answer gate.
- [Evidence and citation model](docs/architecture/evidence-and-citations.md)
  Use for the distinction between candidates, exact reads, Evidence IDs, validated answers, and durable references.
- [Java Language Specification, Java SE 17](https://docs.oracle.com/javase/specs/jls/se17/html/index.html)
  Primary source for language semantics. Use when interview folklore and actual Java behavior disagree.
- [Java Virtual Machine Specification, Java SE 17](https://docs.oracle.com/javase/specs/jvms/se17/html/index.html)
  Primary source for bytecode, runtime data areas, class loading, and execution semantics.
- [Dev.java: Concurrency](https://dev.java/learn/concurrency/)
  Official practical material for threads, synchronization, executors, and concurrent collections.
- [Spring Boot Reference](https://docs.spring.io/spring-boot/reference/)
  Primary source for auto-configuration, configuration, testing, observability, and production behavior.
- [Spring Framework Reference](https://docs.spring.io/spring-framework/reference/)
  Primary source for the IoC container, dependency injection, transactions, MVC, WebFlux, and WebSocket support.
- [Spring Security Reference](https://docs.spring.io/spring-security/reference/)
  Primary source for authentication, authorization, filter chains, method security, and common attack defenses.
- [OpenAI Agents SDK documentation](https://openai.github.io/openai-agents-python/)
  Primary framework reference for the Agent, Runner, tools, sessions, guardrails, tracing, and model loop used by the Python harness.
- [Qdrant documentation](https://qdrant.tech/documentation/)
  Primary source for sparse vectors, filtering, collections, indexing, and operational behavior of the candidate index.
- [Vue documentation](https://vuejs.org/guide/introduction.html)
  Primary source for the Composition API, reactivity, components, and application architecture used by Folio.

## Wisdom (Communities)

- [OpenJDK mailing lists](https://mail.openjdk.org/mailman/listinfo)
  Use for design rationale and expert discussion when JVM or Java behavior goes beyond introductory material.
- [Spring community](https://spring.io/community)
  Use for framework-specific practices and discussions grounded in current Spring behavior.
- [OpenAI Developer Community](https://community.openai.com/)
  Use for real integration failure modes and operational experience; verify claims against official documentation.
