## 每道题怎么背

统一按四句话组织，不背整段：

1. **概念句**：先回答书里的定义。
2. **项目句**：指出 PaperLoom 哪条链路用了它。
3. **取舍句**：为什么这样做，有什么代价。
4. **边界句**：哪些能力项目目前没有，防止被追问时说穿。

例如，面试官问“项目里用过阻塞队列吗”：

> 用过。阻塞队列主要用于生产者和消费者之间的任务缓冲，也可以通过有界容量形成背压。
> PaperLoom 的 `AsyncDiskProductTraceSink` 使用 `ArrayBlockingQueue` 缓冲产品 Trace，由后台线程
> 写磁盘。队列满时使用 `offer` 失败并丢弃 Trace，不阻塞用户请求，所以这里选择的是“业务优先、
> Trace 尽力而为”。代价是 Trace 不是强持久化数据，不能把它说成审计日志。
